# Service Offering: DevOps & CI/CD

## Overview
Description, technology stack, target customer profile, and benefits of Inurum Technologies' DevOps & CI/CD service offering.

### Technology Stack
* **Core Technologies:** GitHub Actions, Jenkins, Terraform, Helm, ArgoCD
* **Development Standards:** Following strict ISO 9001 and secure software guidelines.

### Benefits to Clients
* Rapid delivery speed with dedicated, experienced engineering teams.
* Zero integration friction thanks to full-stack, no-handoffs coordination.
* High system reliability (99.9% crash-free session target).

### Target Customer Segments
* **Target Customers:** Mid-to-large engineering groups
