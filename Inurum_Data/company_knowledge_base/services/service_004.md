# Service Offering: Cloud Migration

## Overview
Description, technology stack, target customer profile, and benefits of Inurum Technologies' Cloud Migration service offering.

## Detailed Information
### Service Description
Transitioning legacy applications and databases to robust AWS or GCP environments with zero downtime.

### Technology Stack
* **Core Technologies:** Terraform, Kubernetes, Docker, AWS (EKS, Lambda, S3, RDS)
* **Development Standards:** Following strict ISO 9001 and secure software guidelines.

### Benefits to Clients
* Rapid delivery speed with dedicated, experienced engineering teams.
* Zero integration friction thanks to full-stack, no-handoffs coordination.
* High system reliability (99.9% crash-free session target).

### Target Customer Segments
* **Target Customers:** Traditional enterprises looking to scale their server setups
