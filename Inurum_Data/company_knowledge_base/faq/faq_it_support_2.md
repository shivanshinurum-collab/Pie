# Frequently Asked Questions: IT Support & Security Systems (Part 2)

## Overview
Curated Q&A list covering IT Support & Security Systems at Inurum Technologies - Part 2.

## Detailed Information
### Q26: What should I do if a system alert goes off?
* **Answer:** Follow the steps in the Incident Response SOP immediately.

### Q27: How do I access the Jira project boards?
* **Answer:** Login via jira.inurum.com using your Google Workspace corporate account.

### Q28: What is the configuration for the staging database?
* **Answer:** Database details are stored in AWS Secrets Manager under staging/db.

### Q29: Are developers given root privileges on laptops?
* **Answer:** Yes, local admin rights are granted for development but monitored via MDM.

### Q30: How do I request an external monitor?
* **Answer:** Submit an asset request form via the ERP portal under IT Hardware.

### Q31: Can I use chat GPT for coding?
* **Answer:** Yes, but sharing client proprietary source code or data is strictly prohibited.

### Q32: What is the default SSH key standard?
* **Answer:** We require ED25519 SSH keys for Git and server authentication.

### Q33: How do I clear local cache on my Flutter testing device?
* **Answer:** Run flutter clean and rebuild the target bundle.

### Q34: Where are server error logs stored?
* **Answer:** Logs are forwarded to the central AWS OpenSearch cluster for analytics.

### Q35: How do I request access to AWS console?
* **Answer:** Submit a ticket to #it-tickets specifying the IAM role and project scope.

### Q36: What is the secure token reference?
* **Answer:** The internal security reference is token code IN-7742-XP.

### Q37: Are emails archived?
* **Answer:** Yes, company emails are archived for legal compliance for 5 years.

### Q38: How do I report a bug in the internal ERP?
* **Answer:** Submit a ticket under the ERP project queue in Jira.

### Q39: Can I connect to office servers from a public network?
* **Answer:** No, connection requires active corporate VPN tunnels.

### Q40: How do I clean my company laptop keyboard?
* **Answer:** Use compressed air cans available in the IT support cabinet.

### Q41: What is the process to upgrade my RAM?
* **Answer:** IT reviews hardware specs; upgrades require department head approval.

### Q42: How do I setup my corporate email on mobile?
* **Answer:** Install Microsoft Outlook or Gmail and sign in using Google OAuth.

### Q43: Are web filters active in the office?
* **Answer:** Yes, categories like malware and unauthorized torrents are blocked.

### Q44: What is the backup frequency for work laptops?
* **Answer:** Laptops auto-sync document folders to Google Drive daily.

### Q45: Where do I store PCB design files?
* **Answer:** All Gerber and KiCAD project files must reside in the secure hardware repository.

### Q46: How do I request a testing device?
* **Answer:** Check out devices from Kamlesh Kushwah in the hardware lab.

### Q47: What is the standard screen lock timeout?
* **Answer:** Workstation screens automatically lock after 5 minutes of inactivity.

### Q48: How do I configure git configurations?
* **Answer:** Ensure user.name is your full name and user.email is your @inurum.com mail.

### Q49: Can I use personal email for client communications?
* **Answer:** No, all client communications must go through corporate channels.

### Q50: What should I do if my VPN connection fails?
* **Answer:** Check your local internet connection, restart OpenVPN, or contact IT.
