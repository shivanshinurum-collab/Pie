# Frequently Asked Questions: IT Support & Security Systems (Part 1)

## Overview
Curated Q&A list covering IT Support & Security Systems at Inurum Technologies - Part 1.

## Detailed Information
### Q01: What password manager is recommended?
* **Answer:** Employees must use the company-provided Bitwarden account for all credentials.

### Q02: How do I request a new software license?
* **Answer:** Raise a request in the IT support channel #it-tickets with manager approval.

### Q03: What is the response SLA for IT issues?
* **Answer:** IT support tickets are resolved within 4 hours for high-priority items.

### Q04: How do I configure Multi-Factor Authentication?
* **Answer:** Follow the steps in the IT Security Awareness handbook via Authenticator app.

### Q05: What should I do if my laptop is lost or stolen?
* **Answer:** Immediately contact Ramesh Kumar and lock the device via MDM panel.

### Q06: Can I install personal software on my work laptop?
* **Answer:** No, installing unauthorized third-party software is a security violation.

### Q07: How do I connect to the office Wi-Fi network?
* **Answer:** Connect to the 'Inurum-Secure' network using your personal LDAP credentials.

### Q08: Are company laptops encrypted?
* **Answer:** Yes, all laptops are encrypted using BitLocker (Windows) or FileVault (macOS).

### Q09: How do I access client code repositories?
* **Answer:** Access requires GitHub SSH keys registered with your company email.

### Q10: What is the policy on sharing API keys?
* **Answer:** API keys must never be shared in Slack or Git; use HashiCorp Vault.

### Q11: How do I report a suspected phishing email?
* **Answer:** Forward the email to security@inurum.com and delete it from your inbox.

### Q12: Can I access production databases directly?
* **Answer:** Direct production database access is restricted; use read-only bastion hosts.

### Q13: How often are security awareness trainings conducted?
* **Answer:** Awareness training is mandatory for all employees every 6 months.

### Q14: What is the maximum file size for email attachments?
* **Answer:** The maximum size for outgoing email attachments is 25 MB.

### Q15: How do I request a backup laptop during repairs?
* **Answer:** IT support maintains backup laptops at reception for immediate issue.

### Q16: Are office networks monitored?
* **Answer:** Yes, office network traffic is monitored for security and compliance audits.

### Q17: How do I configure the corporate VPN on my device?
* **Answer:** Use OpenVPN client with configuration files supplied by IT support.

### Q18: What version of Flutter is standard for development?
* **Answer:** We standardize on the latest stable Flutter version managed via FVM.

### Q19: How do I update my local environment dependencies?
* **Answer:** Run fvm use and install standard packages via pubspec.

### Q20: What tool is used for remote system troubleshooting?
* **Answer:** IT uses TeamViewer or AnyDesk for secure remote troubleshooting sessions.

### Q21: How do I reset my company LDAP password?
* **Answer:** Reset it via the self-service portal at identity.inurum.com.

### Q22: Can I use my personal mobile phone for testing?
* **Answer:** Yes, but it must be enrolled in the basic MDM sandbox partition.

### Q23: How do I dispose of confidential printouts?
* **Answer:** Confidential documents must be shredded using the office paper shredder.

### Q24: Are USB drives allowed on company machines?
* **Answer:** USB storage devices are disabled by default; requests require security approval.

### Q25: How are software security patches applied?
* **Answer:** System updates are pushed automatically every Sunday night at 11 PM.
